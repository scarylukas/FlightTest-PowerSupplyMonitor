# ⚡ Power Supply Monitor (macOS)

Control and monitor your laboratory power supply directly from macOS.

---

## 💡 About

Power Supply Monitor is a native macOS application designed to work with USB-connected laboratory power supplies.

Many popular devices (such as Korad) are shipped with Windows-only software.  
This app provides a clean and simple macOS alternative — no virtual machines, no workarounds.

---

## ⚡ Features
- Real-time voltage and current monitoring  
- Control output parameters directly from macOS  
- Clean, native macOS interface  
- Lightweight and stable  

---

## 💻 Requirements
- macOS 10.15 or newer  
- Compatible USB laboratory power supply  (ex. Korad)
